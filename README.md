# 记忆卡牌配对游戏

一个精美的记忆卡牌配对游戏，带有嘴碎小助手互动功能。

## 功能特性

### 🎮 游戏玩法
- **4×4 简单模式**：16张卡牌，8对配对
- **5×5 困难模式**：25张卡牌，12对配对 + 1张特殊卡
- 翻转卡牌寻找相同的配对
- 记录游戏时间、步数和最佳成绩
- 支持深色/浅色主题切换

### 🤖 嘴碎小助手
- 位于页面左侧的智能助手
- 根据游戏情况实时互动
- 丰富的台词库（46条）
  - 游戏开始：开场白
  - 配对失败：调皮嘲讽（15条）
  - 配对成功：热情鼓励（15条）
  - 连续成功：庆祝台词（5条）
  - 连续失败：安慰台词（3条）
  - 游戏胜利：祝贺台词（5条）
- 打字机效果显示台词
- 智能防重复机制（避免短时间内重复）

### 🎨 视觉效果
- 精美的渐变卡牌设计
- 流畅的3D翻转动画
- 匹配成功粒子特效
- 响应式设计，支持移动端

### 🔊 音效系统
- 翻牌音效
- 配对成功/失败音效
- 胜利庆祝音效
- 可开关的音效控制

## 如何使用

### 打开游戏
直接用浏览器打开 `memory-card-game.html` 文件即可开始游戏。

### 操作说明
1. **选择难度**：点击"4×4 简单"或"5×5 困难"按钮
2. **开始游戏**：点击卡牌翻转，寻找相同的配对
3. **查看进度**：顶部显示游戏时间、步数和最佳成绩
4. **重新开始**：点击"重新开始"按钮重新开始游戏
5. **切换主题**：点击月亮/太阳图标切换深色/浅色主题
6. **控制音效**：点击音量图标开启/关闭音效

### 游戏规则
- 每次只能翻转两张卡牌
- 两张卡牌相同则配对成功，保持翻转状态
- 两张卡牌不同则配对失败，1秒后自动翻回
- 成功配对所有卡牌即可获胜
- 步数越少、时间越短，成绩越好

- CSS3实现3D翻转和动画效果

- 支持主流现代浏览器
- 包含webkit、moz、ms等浏览器前缀

### 数据存储
- 使用localStorage保存最佳成绩
- 使用localStorage保存主题偏好
- 5×5模式自动保存最佳记录
